## Description

back-end usign NestJS, TypeORM and GraphQL.


Relation
User * <---> * Messages


You can use yarn or npm to run.


To access the GraphQL Playground:
http://localhost:3333/graphql

## Installation

```bash
$ npm install
$ yarn install
```

## Running the app

```bash
# development
$ npm run start
$ yarn start

# watch mode
$ npm run start:dev
$ yarn start --watch

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
